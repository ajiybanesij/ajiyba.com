    About

    This service provides decentralized applications with a secure file storage solution as a API.


    Essentially, developers can take the service's unique secret key and use it to send files to the service.


    The service encrypts and uploads the file to IPFS or AWS S3. If developers wish to receive the file, they can decrypt it using the secret key and then download it.
